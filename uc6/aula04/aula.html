Aula 04 - Estruturas de Repetição

1. While Loop

O loop while é utilizado quando queremos executar um bloco de código enquanto uma condição for verdadeira.

                    
            var contadorWhile = 0;
    
            while (contadorWhile < 5) {
                console.log("Interação While: " + contadorWhile);
                contadorWhile++;
                
            }

2. Do-While Loop

O loop do-while é semelhante ao while, mas a condição é verificada após a execução do bloco de código, garantindo que o bloco será executado pelo menos uma vez.

        
        var contadorDoWhile = 0;

        do {
            console.log("Iteração Do-While: " + contadorDoWhile);
            contadorDoWhile++;
        } while (contadorDoWhile < 5);

3. For Loop

O loop for é utilizado quando sabemos antecipadamente quantas vezes o bloco de código deve ser executado.

        
            for (var i = 0; i < 5; i++) {
                console.log("Iteração For: " + i);
            }

4. For-In Loop

O loop for-in é usado para iterar sobre as propriedades de um objeto.

        
        var carro = { marca: "Toyota", modelo: "Corolla", ano: 2022 };

        for (var propriedade in carro) {
            console.log(propriedade + ": " + carro[propriedade]);
        }

5. For-Of Loop

O loop for-of é usado para iterar sobre elementos iteráveis ​​como arrays e strings.

        
            var frutas = ["Maçã", "Banana", "Morango"];

            for (var fruta of frutas) {
                console.log("Fruta: " + fruta);
            }
