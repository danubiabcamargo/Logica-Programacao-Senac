Aula 06 - Introdução a Funções em JavaScript

O que são Funções?

Funções são blocos de código reutilizáveis que podem ser definidos e chamados em diferentes partes do programa. Elas facilitam a organização e modularização do código.

Declaração de Funções

Uma função em JavaScript pode ser declarada da seguinte forma:

        
            // Declaração de uma função simples
            function saudacao() {
                alert('Olá, seja bem-vindo!');
            }

Chamada de Funções

Uma vez que uma função é declarada, ela pode ser chamada/executada em qualquer parte do código:

        
            // Chamando a função saudacao
            saudacao();

Parâmetros e Retorno de Funções

Funções podem receber parâmetros (dados de entrada) e retornar valores (dados de saída):

        
            // Função com parâmetros e retorno
            function soma(a, b) {
                return a + b;
            }

            // Chamando a função soma
            var resultado = soma(3, 5);
            console.log('Resultado da soma:', resultado);

Escopo de Variáveis

Funções também possuem escopo, o que significa que as variáveis declaradas dentro de uma função não são acessíveis fora dela:

        
            // Função com variável local
            function exibirMensagem() {
                var mensagem = 'Essa é uma mensagem local.';
                console.log(mensagem);
            }

            // Tentativa de acessar a variável fora da função (resultará em erro)
            console.log(mensagem);

Exemplo Prático

Vamos criar uma função que calcula a média de uma lista de números:

        
            function calcularMedia(lista) {
                var soma = 0;
                for (var i = 0; i < lista.length; i++) {
                    soma += lista[i];
                }
                return soma / lista.length;
            }

            var numeros = [5, 8, 12, 3, 9];
            var media = calcularMedia(numeros);
            console.log('Média dos números:', media);
