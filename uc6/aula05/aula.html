Aula 05 - Trabalhando com Listas em JavaScript

Declaração de Listas

JavaScript oferece a capacidade de trabalhar com listas, uma estrutura que permite armazenar e manipular conjuntos de dados.

        
            // Declaração e inicialização de uma lista
            var frutas = ['Maçã', 'Banana', 'Uva'];

Acessando Elementos em uma Lista

Os elementos em uma lista podem ser acessados por meio de índices.

        
            // Acessando o primeiro elemento da lista
            var primeiraFruta = frutas[0];

            // Acessando o segundo elemento da lista
            var segundaFruta = frutas[1];

Adicionando e Removendo Elementos

É possível adicionar e remover elementos em uma lista de diversas maneiras.

        
            // Adicionando um elemento ao final da lista
            frutas.push('Pêra');

            // Removendo o último elemento da lista
            frutas.pop();

            // Adicionando um elemento no início da lista
            frutas.unshift('Abacaxi');

            // Removendo o primeiro elemento da lista
            frutas.shift();

Exemplo Prático

A seguir, um exemplo prático de como trabalhar com listas em JavaScript:

        
            // Declaração e inicialização de uma lista
            let frutas = ['Maçã', 'Banana', 'Uva'];

            // Exibindo os elementos da lista
            document.write("<h3>Frutas:</h3>");
            document.write("<ul>");
            for (let i = 0; i < frutas.length; i++) {
                document.write("<li>" + frutas[i] + "</li>");
            }
            document.write("</ul>");

            // Adicionando um novo elemento à lista
            frutas.push('Pêra');

            // Exibindo a lista atualizada
            document.write("<h3>Frutas Atualizadas:</h3>");
            document.write("<ul>");
            for (let i = 0; i < frutas.length; i++) {
                document.write("<li>" + frutas[i] + "</li>");
            }
            document.write("</ul>");
